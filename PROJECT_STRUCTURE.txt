🚀 Quick Start
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Run quick start script:
   ./quickstart.sh

2. Update .env.local with your database URL

3. Start development server:
   npm run dev

4. Open http://localhost:3000
